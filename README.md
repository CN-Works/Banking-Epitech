<div align='center'><h1>Banking Epitech</h1></div>

A simple banking application for Fivem purposes.
<br/>
<div align='left'><h1>Features</h1></div>

- Accéder à l'interface via un point ou un objet (ATM)
- Voir son compte et ses paiements et tranferts.
- Retirer de l'argent liquide
- Déposer de l'argent liquide.
- Envoyer de l'argent à un autre joueur connecté.
- Créer une API pour ajouter des paiements.
